运算符 异或^

JS中的异或运算符^主要是在位运算的时候使用。在普通的数使用时有一个转换过程，即将二进制的数转换为十进制。

其中原理——离散数学的异或运算性质：

  • 交换律：a ^ b ^ c <=> a ^ c ^ b
  • 任何数于0异或为任何数： 0 ^ n => n
  • 相同的数异或为0：n ^ n => 0
