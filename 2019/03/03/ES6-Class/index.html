ES6 Class

2019-03-03 前端 PV:

ES6始，Class成为标准，其更加符合面向对象的代码风格优势显得尤为突出，本篇介绍和比较JS构造函数与Class各自特性与区别。

JS构造函数

Class

语法糖

由此可见Class本质是一个语法糖，它实际上仍然使用的是JS的原型来完成继承。

继承

  • ES6之前 继承通过将低级构造函数的原型赋值为高级函数的实例来实现
    Dog.prototype = new Animal()
  • ES6 Class 在定义Class的时候使用extends来实现继承
    class Dog extends Animal
    1. 如果在定义Class时使用了继承extends，则必须有Super
    2. Super是将被继承class中的constructor执行一遍

总结

  • Class语法上更贴合面向对象的写法
  • Class实现继承更加易读、易理解
  • 更易于写java等后端语言的使用
  • 更符合学生课堂学习内容
  • Class本质是语法糖，使用的是构造函数中的prototype
